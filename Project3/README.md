# Customer Segmentation using K-Means Clustering

## Project Overview
This project segments customers into different groups using K-Means Clustering and PCA. It helps businesses identify customer groups for targeted marketing.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Algorithms
- PCA
- K-Means Clustering
- Elbow Method
- Silhouette Score

## Project Workflow
1. Import dataset
2. Data preprocessing
3. Feature scaling
4. PCA
5. Elbow Method
6. Silhouette Score
7. K-Means Clustering
8. Visualization
9. Cluster Analysis

## Results
The customers were successfully divided into clusters based on their characteristics, helping identify different customer segments.
